  • 2024, Balance It: A Game for CFD

    • Project GitHub Page

    This is the final project of the track "Computational Fluid Dynamics" in 0-1 Winter Camp 2024, Shenzhen. It is an interactive game that simulates the process of balancing a ball in wind using CFD. The game is implemented from scratch using Python.

  • 2023, HTTP Server from Scratch

    • Project GitHub Page

    This is the final project for CS305: Computer Networks 2023 Fall. I implemented a simple HTTP server from scratch using Python. The server can handle GET and POST requests, and can serve static files and dynamic pages.

  • 2023, Task Manager: I Need To

    • Project GitHub Page
